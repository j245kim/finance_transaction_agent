# STFO Financial Advisor 사용법

당신의 AI 금융 가이드.

AI 기능을 활용하여, 각종 금융 관련 질문부터 투자까지 가능성을 탐험해 보세요.

- 투자 성향 분석은 어떤 방식으로 적용되나요?
    
    한국의 5대 은행이 사용하는 방식과 동일한 방식에 우리만의 최적화된 점수 계산을 추가로 적용하여 사용자의 투자 성향을 분석합니다. 분석 결과는 우리의 데이터 베이스에 저장되며, 답변을 생성할 때 데이터로만 활용됩니다.
    
- 최신 금융 데이터는 어떻게 불러오나요?

    - 1️⃣ **멀티프로세싱 작업**
        - 크롤링해야 할 **웹사이트 개수만큼 프로세스** 생성
        - 각 프로세스는 **별도의 웹사이트를 담당**하여 동작
    - 2️⃣ **멀티스레딩 작업**
        - 각 프로세스는 **웹사이트의 카테고리별로 스레드**를 생성
        - 병렬 작업을 통해 크롤링 속도 최적화
    - 3️⃣ **무작위 지연시간 적용**
        - 적절한 딜레이로 실시간 데이터 수집으로 관련한 서버 부하 방지!
    - 4️⃣ **MongoDB 저장**

        ### OLS 회귀 분석 결과 해석 기준 (중요한 값 정리본)
        
        ---
        
        ### **모델의 설명력**
        
        - **R-squared (결정 계수)**
            - **설명**: 모델이 종속 변수의 변동성을 설명합니다.
            - **해석**: 값이 1에 가까울수록 모델이 데이터를 잘 설명한다고 볼 수 있음. 값이 낮으면 설명력이 부족한 모델로 판단합니다.
        
        ---
        
        ### **모델의 유의미성**
        
        - **F-statistic**
            - **설명**: 모델 전체가 종속 변수를 설명하는지 확인하는 지표입니다.
            - **해석**: F-statistic의 **P-value**가 0.05 이하일 때, 모델이 유의미하다고 판단합니다.
        
        ---
        
        ### **각 계수의 유의미성**
        
        - **P-value**
            - **설명**: 각 회귀 계수가 통계적으로 유의미한지 검정하는 값입니다.
            - **해석**: **P-value**가 **0.05**보다 작으면 해당 계수는 유의미하다고 판단합니다.
        
        ---
        
        ### 파마의 5요인 펙터 (만약에 모델의 설명력이 크다면)
        
        - **Mkt-RF (시장 위험 프리미엄)**
            - 시장 전체의 수익률에서 무위험 자산 수익률을 뺀 값입니다.
            - **기준**: 시장 수익률이 높을수록 해당 값은 커지며, **양의 값**이면 시장이 상승하는 경향이 있다는 의미입니다.
        - **SMB (Small Minus Big)**
            - 작은 기업의 수익률에서 큰 기업의 수익률을 뺀 값입니다.
            - **기준**: **양의 값**이면 작은 기업들이 큰 기업들보다 더 높은 수익률을 보인다는 의미입니다. 반대로 **음의 값**은 큰 기업들이 더 나은 성과를 보였다는 뜻입니다.
        - **HML (High Minus Low)**
            - 가치주(저 P/E, 저 P/B)와 성장주(고 P/E, 고 P/B)의 수익률 차이입니다.
            - **기준**: **양의 값**이면 가치주가 성장주보다 더 나은 성과를 보였다는 의미입니다.
        - **RMW (Robust Minus Weak)**
            - 수익성이 높은 기업(강한 기업)과 수익성이 낮은 기업(약한 기업)의 수익률 차이입니다.
            - **기준**: **양의 값**이면 수익성이 높은 기업들이 더 높은 수익률을 기록했다는 의미입니다.
        - **CMA (Conservative Minus Aggressive)**
            - 보수적인 투자 성향을 가진 기업과 공격적인 투자 성향을 가진 기업의 수익률 차이입니다.
            - **기준**: **양의 값**이면 보수적인 기업들이 더 높은 수익을 올렸다는 의미입니다.

 **민감한 투자 주제로 대화를 나눠도 괜찮을까요?**

우리는 강력한 **보안 표준**을 적용하여, 이용자의 정보를 안전하게 보호합니다.

- **강력한 보안 시스템**
    
    모든 정보는 **소비자보호법을 준수**하며, **고급 암호화 알고리즘**을 통해 안전하게 저장됩니다.
    
- **개인정보 관리**
    
    이용자는 언제든지 자신의 개인정보를 **조회, 수정, 삭제**할 수 있습니다.
    
    회사는 오직 **서비스 제공 목적**으로만 이용자의 개인정보를 사용합니다.
    
- **STFO DB 보관 항목**
    
    모든 데이터는 **STFO의 보안 DB**에 저장됩니다. 여기에는 다음과 같은 정보가 포함됩니다:
    
    📌 사용자 아이디
